# 📊 Telecom Customer Churn Analysis

## Overview

This project predicts customer churn in a telecom company using Machine Learning and identifies customers who are likely to leave and provides business recommendations to improve customer retention.



## Project Objectives

- Predict customer churn
- Perform SQL-based customer analysis
- Build a Logistic Regression model
- Explain predictions using SHAP
- Segment customers into At Risk, Loyal, and Dormant groups
